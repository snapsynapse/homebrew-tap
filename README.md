# Snap Synapse Homebrew Tap

Homebrew formulae for Snap Synapse tools.

## Install

```bash
brew install snapsynapse/tap/harnessie
```

## Formulae

- [agentlink](https://github.com/snapsynapse/agentlink) — Sync one AGENTS.md to every AI coding tool with symlinks, no codegen. Site: https://agentlink.run/
- [guidecheck](https://github.com/snapsynapse/guidecheck) — Local verifier for GuideCheck Human-Verifiable Assistant Guides (`guidecheck-verify`). Site: https://guidecheck.org/
- [harnessie](https://github.com/snapsynapse/harnessie) - Brain-agnostic multi-agent harness with verification gates, containment-aware routing, and a tamper-evident audit log. Site: https://harnessie.com/
- [prompter-kit](https://github.com/snapsynapse/prompter-kit) — Backup, restore, and manage Elgato Prompter scripts from the command-line. Site: https://prompterkit.app/
- [substack2md](https://github.com/snapsynapse/substack2md) — Export Substack newsletters to clean Markdown with YAML frontmatter.
